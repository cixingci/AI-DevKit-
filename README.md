## 📝 API端点列表

### 基础管理

- `GET /api/v1/projects` - 获取项目列表
- `POST /api/v1/projects` - 创建项目
- `DELETE /api/v1/projects/{id}` - 删除项目

### 卡片管理

- `GET /api/v1/cards` - 获取卡片列表
- `POST /api/v1/cards` - 创建卡片
- `PUT /api/v1/cards/{id}` - 更新卡片
- `DELETE /api/v1/cards/{id}` - 删除卡片
- `GET /api/v1/cards/{id}` - 获取卡片详情

### AI服务

- `POST /api/v1/ai/llm/configs` - 创建AI配置
- `GET /api/v1/ai/llm/configs` - 获取配置列表
- `POST /api/v1/ai/generate` - AI内容生成
- `POST /api/v1/ai/llm/configs/{id}/test` - 测试连接

### 工作流管理

- `POST /api/v1/ai/workflows/execute` - 执行工作流
- `GET /api/v1/ai/workflows/runs` - 获取运行记录
- `GET /api/v1/ai/workflows/project/{id}/initialize` - 项目初始化
