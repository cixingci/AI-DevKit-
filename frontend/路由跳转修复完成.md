## 📝 技术细节

### 为什么使用 router-link?
```vue
<!-- 问题：直接使用 el-menu-item -->
<el-menu-item index="/projects">
  <span>项目管理</span>
</el-menu-item>

<!-- 解决方案：使用 router-link -->
<router-link to="/projects" class="menu-link">
  <el-menu-item index="/projects" @click.prevent>
    <span>项目管理</span>
  </el-menu-item>
</router-link>
```

### 为什么添加 @click.prevent?
```javascript
// 防止 el-menu-item 的默认点击行为
<el-menu-item @click.prevent>
  <span>项目管理</span>
</el-menu-item>
```

### 为什么需要 active-class?
```css
.menu-link.active .el-menu-item {
  color: #409eff;
  background-color: #ecf5ff;
}
```

确保当前激活的菜单项有正确的视觉反馈。
