## 🎯 技术要点

### 为什么使用 router 属性？
```vue
<!-- router 属性告诉 Element Plus 使用 Vue Router -->
<el-menu router>

<!-- 现在 el-menu-item 的 index 属性会自动处理路由跳转 -->
<el-menu-item index="/cards">
  <span>卡片视图</span>
</el-menu-item>
```

### 为什么按钮宽度很重要？
```css
/* 太宽会占用过多空间 */
.nav-menu-items .el-menu-item {
  min-width: 100px;  /* 太宽 */
  max-width: 70px;   /* 限制文字 */
}

/* 合适的宽度 */
.nav-menu-items .el-menu-item {
  min-width: 80px;   /* 合适 */
  flex: 0 0 auto;    /* 自动调整 */
}

.nav-menu-items .el-menu-item span {
  max-width: 50px;   /* 限制文字 */
}
```
