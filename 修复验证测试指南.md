## 🐛 常见问题

### 问题1: 下拉框不显示

**可能原因**:
- API 调用失败
- 卡片类型数据为空
- 浏览器缓存问题

**解决方案**:
```bash
# 1. 检查 API 是否正常
curl http://localhost:8888/api/v1/card-types

# 2. 清除浏览器缓存
按 Ctrl + Shift + Delete

# 3. 硬刷新页面
按 Ctrl + Shift + R
```

### 问题2: 选择卡片类型后无反应

**可能原因**:
- JavaScript 错误
- Vue 状态管理问题
- API 响应格式问题

**解决方案**:
```bash
# 1. 检查控制台错误
按 F12 查看错误信息

# 2. 检查 API 响应格式
curl http://localhost:8888/api/v1/card-types

# 3. 重启前端服务
# 在前端终端按 Ctrl+C 停止
# npm run dev
```

### 问题3: 卡片类型列表为空

**可能原因**:
- 数据库中没有卡片类型
- API 返回空数组

**解决方案**:
```bash
# 1. 检查数据库
curl http://localhost:8888/api/v1/card-types

# 2. 创建卡片类型
# 访问 /card-types 页面创建新类型

# 3. 刷新页面
按 Ctrl + Shift + R
```
